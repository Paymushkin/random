# Рандомайзер для мероприятий

Веб-приложение для случайного выбора победителей на мероприятиях Central Warehouses Uzbekistan.

## Функционал

- **Два режима работы:**
  - Выбор случайных имен из списка участников
  - Выбор случайных чисел из заданного диапазона

- **Гибкие настройки:**
  - Количество победителей в каждой итерации
  - Режим исключения уже выбранных победителей
  - История всех выборов с временными метками

- **Простой интерфейс:**
  - Переключение между режимами одной кнопкой
  - Анимированное отображение результатов
  - Возможность сброса истории

## Использование

### Режим выбора имен:
1. Введите список участников (каждое имя с новой строки)
2. Укажите количество победителей
3. Нажмите "Выбрать победителей"

### Режим выбора чисел:
1. Задайте диапазон (от - до)
2. Укажите количество победителей
3. Нажмите "Выбрать победителей"

### Дополнительные возможности:
- **Исключать уже выбранных:** При включенной опции повторный выбор будет исключать предыдущих победителей
- **Сбросить историю:** Очистить всю историю выборов и список победителей

## Развертывание на GitHub Pages

1. Перейдите в Settings → Pages
2. В разделе "Source" выберите ветку `master` и папку `/ (root)`
3. Сохраните изменения
4. Приложение будет доступно по адресу: `https://[ваш-username].github.io/random/`
